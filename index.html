About

UX | UI | FRONTEND

I am studying Multimedia Design at Copenhagen School of Design and Technology, with main focus on UI|UX & Frontend Design.
I am a person driven by visuals with an eye for detail. I am passionate about learning and building new things. I like to design and code minimalist user interfaces and make simple games.
I made this website to showcase some of my favourite projects, outline tools I usually work with and skills I have gained so far on my journey.

Projects

Brand Identity & Mobile Prototype

This project's objective was to establish effective visual communication for a brand and designing seamless purchase flow for a chosen type of e-commerce. I worked with record store specializing in vintage vinyls. Logo was made in Illustrator and prototyping done in XD.

Character/Scene Design & Animation

Composter is a simple one level game made in JavaScript. I worked with principles of flat design and simple shapes when developing the character and scenes. Main objective of this project was to become more familiar with animation in JavaScript, as well as, planning and design of the program in accordance with the game logic.

SVG & GSAP Animation

Main objective of this project was to design and code product configurator website. My main responsibilities were wireframing and coding of the configurator product page. It is configurable in different colors with different components, some of which are mutually exclusive. Part of the learning goal was to work with an manipulate SVG files and also becoming familiar with the GSAP animations.

User Research & User Testing

Aim of this project was to establish a cooperation with an outside client. Our main goal was to participate on building of client's online presence, communication of values, while introducing the product. I was in charge of user research and testing, mainly focused on conducting surveys, interviews and tests that provided us with insights about users' behaviour and navigate our design process towards final solution.

User Input & Neumorphic Design

I choose this project in order to practice different ways of handling user inputs and switch statements, as well as get familiar with the usage of JavaScript classes in code organization. From a visual point of view I worked with neumorphic design style that uses homogenous color set up with light and dark shadows creates 3D effect.

JSON Data Handling, Sort, Filter & Search

This project is a representaton of a student administration interface. Its main focus was data clean up and its organization within the interface in different ways. Students can be filtered by house, gender, roles and student status, while being sorted alphabetically. Under some circumstances students can be given different roles or be expelled.
